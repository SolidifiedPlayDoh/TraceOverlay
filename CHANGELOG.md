# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2025-02-16

### Added

- Full-screen transparent overlay for image tracing
- Menu bar control (open image, lock/unlock, opacity)
- Move image by dragging, zoom with scroll wheel, rotate with Option + drag
- Lock mode: click-through overlay so you can draw in the app behind
- Opacity presets: 30%, 50%, 70%, 100%
- Drag & drop image onto screen, or Open Image… (⌘O)
- Support for PNG, JPEG, GIF, TIFF, BMP
- Build from source with `./build.sh` (no Xcode project)
